# Installation Guide

1. Install Termux from F-Droid or the official Termux GitHub releases.
2. Open Termux and run:

```bash
pkg update -y
pkg install git -y
git clone https://github.com/Mraprguild/termux-os.git
cd termux-os
chmod +x install.sh
./install.sh
```

Restart Termux or run `source ~/.bashrc` after installation.
