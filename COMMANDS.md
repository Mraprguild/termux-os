# Command Reference

| Command | Description |
|---|---|
| `termux-os help` | Display available commands |
| `termux-os info` | Show device and Termux details |
| `termux-os backup` | Back up the Termux home directory |
| `termux-os restore FILE` | Restore a backup archive |
| `termux-os update` | Update packages and project files |
| `termux-os dev` | Install development toolchains |
| `termux-os linux` | Install Ubuntu or Debian |
